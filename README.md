# AgriSmart 🌾

**AgriSmart** est une solution complète de suivi et de prédiction des prix des produits agricoles au Sénégal. Elle permet aux agriculteurs et aux commerçants de suivre l'évolution du marché en temps réel et de prendre des décisions éclairées grâce à l'IA.

## Structure du Projet

- **`flutter_app/`** : Application mobile développée avec Flutter pour Android, iOS et Web.
- **`backend/`** : API REST développée avec Django et Django REST Framework.
